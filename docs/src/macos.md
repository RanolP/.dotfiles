# macOS

## System Defaults

| Setting | Value |
|---------|-------|
| Dock | Auto-hide, no recent apps, bottom-left hot corner → lock screen (⌘) |
| Finder | Show all files/extensions, column view, path bar |
| Appearance | Dark mode |
| Key repeat | Rate 2, initial delay 15 |
| Press-and-hold | Disabled (key repeat instead of accent popup) |
| Fn keys | F1–F12 by default (no need to hold Fn) |
| Trackpad | Tap to click, three-finger drag |
| Spotlight | Disabled (Cmd+Space → Raycast) |
| Input method | F18 → 한영 toggle |
| Liquid Glass | Disabled (macOS 26 Tahoe) |

## Security

- Touch ID for `sudo`
- GPG commit signing

## Chrome Policy

Bitwarden extension force-installed via `/Library/Google/Chrome/policies/managed/extensions.json`.
