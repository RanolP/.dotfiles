# GUI Apps

Installed via Homebrew casks, managed by nix-darwin.

| App | Purpose |
|-----|---------|
| Ghostty | Terminal emulator |
| Raycast | Launcher (replaces Spotlight) |
| Karabiner-Elements | Keyboard remapping (F18 → 한영) |
| LinearMouse | Mouse/trackpad customization |
| Firefox Developer Edition | Browser (symlinked to `~/Applications` for macOS privacy access) |
| Chrome | Browser (forced Bitwarden extension via enterprise policy) |
| Notion | Notes |
| Slack | Messaging |
| Discord | Messaging |
| Figma | Design |
| Bitwarden | Password manager |
| Keybase | Encrypted messaging / file storage |
| Syncthing | File sync |
| Claude | Anthropic Claude desktop app |
| Android Command Line Tools + Temurin | Android development |

### Fonts

- **Iosevka Nerd Font** — terminal / editor
- **Pretendard** — UI / Korean text
