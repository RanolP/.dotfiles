### Extra CLI (Homebrew brews)

- **mise** — tool version manager
- **git-absorb** — auto-fixup commits
- **git-filter-repo** — rewrite git history

### Nix packages

- **gnupg** + **pinentry_mac** — GPG signing
- **nix-your-shell** — nushell integration for `nix develop`
- **xcodes** — Xcode version manager (prebuilt binary; brew formula requires Xcode to build Xcode)
