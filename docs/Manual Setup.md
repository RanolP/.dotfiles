# Manual Setups

## Cursor

- Login to cursor

### First-launch

- Use cursor command only

## Bitwarden

- Vault timeout: Never
- Check Unlock with Touch ID
- Check Show menu bar icon
- Check Close to menu bar
- Check Start to menu bar
- Check Start automatically on login

## Slack

- Login if you're work

## Macbook

- Set default web browser as Firefox Developer Edition

## Firefox

- Enable vertical tab
- Set devtool "Dock to Right"

### Extensions

- Firefox Multi-Account Containers

### GitHub CLI

- `gh auth login --scopes read:packages`

### Safari

- Settings
  - Advanced > Show features for web developers
